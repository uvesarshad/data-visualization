# **App Name**: DataSense

## Core Features:

- Data Source Integration: Allow users to upload CSV and Excel files, or connect directly to SQL databases for data input.
- Intelligent Visualization Generation: Automatically analyze uploaded data and generate appropriate graphs, charts, and data visualizations for an interactive dashboard.
- Interactive Dashboard Display: Present dynamically generated visualizations in a customizable and interactive dashboard, supporting actions like zooming, panning, and filtering data.
- AI-Powered Data Insights & Predictions: Utilize a Gemini-based reasoning and grounding tool to generate detailed insights and predictions from the data, with a user-toggleable on/off switch.
- Raw Data Viewer: Provide a clean, tabular view of the underlying raw data imported from various sources.
- Dashboard Layout Adjustment: Enable users to rearrange, resize, and configure visualization panels within the dashboard for a personalized view.

## Style Guidelines:

- A sophisticated dark color scheme is chosen to provide a professional backdrop that enhances data readability and highlights visualizations. The background color is a deep, desaturated blue-grey (#141A1F).
- The primary color is a vibrant yet professional blue (#1A80E6), symbolizing clarity, technology, and trust, ideal for highlighting key data points and interactive elements.
- An accent color of medium violet (#5E5EE6) is used for call-to-action buttons, interactive states, and distinct dashboard components, providing an energetic contrast to the primary blue while maintaining visual harmony.
- Headline font: 'Space Grotesk', a modern sans-serif with a techy feel, perfect for titles and short, impactful text in a data-focused application. Body text font: 'Inter', a clean and highly readable sans-serif for numerical data, detailed insights, and general text.
- Utilize a consistent set of clean, minimalist line icons for navigation, data actions (e.g., upload, save, refresh), and chart types, ensuring they are easily recognizable against the dark background.
- Implement a modular and responsive grid-based layout that dynamically adjusts to different screen sizes, allowing for flexible arrangement and clear presentation of multiple interactive data visualizations.
- Incorporate subtle and smooth transition animations for loading data, generating new charts, and expanding/collapsing dashboard panels to enhance the user experience without being distracting.
